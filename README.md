## Connect 4 :game_die:

The rules of the game are simple: the first player to connect 4 of the same colored discs in a row (either vertically, horizontally, or diagonally) wins.

## Getting Started :checkered_flag:

> install dependences
```bash
yarn
``` 

> run the application
```bash
yarn start
``` 

> run the tests
```bash
yarn test
``` 

> run the e2e tests
```bash
yarn test:e2e
```
